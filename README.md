# 🧬 Bioinformatics Agent

An interactive AI-powered bioinformatics analysis platform built using Streamlit.

## 🚀 Features

* 📊 Statistical Analysis (t-test, ANOVA, Wilcoxon, Chi-square, correlation)
* 🧬 MSI Scoring (Microsatellite Instability)
* 🧪 Differential Gene Expression (DGE)
* 🌋 Volcano Plots
* 🧬 Functional Enrichment (GO + KEGG)
* ⏳ Survival Analysis (Kaplan-Meier)
* 📈 Interactive Visualizations (Plotly)
* 🤖 AI-powered biological interpretation (Ollama)

## 📂 Supported Data Formats

* CSV
* TSV
* Excel (.xlsx)
* PDF
* Can upload files uder 200MB

## 🖥️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/Bioinformatics-Agent.git
cd Bioinformatics-Agent

pip install -r requirements.txt
streamlit run app.py
```

## 🧪 Use Cases

* Cancer genomics (MSI analysis)
* Gene expression analysis
* Multi-omics data exploration
* Research data visualization

## 🧠 AI Integration

Uses local LLM via Ollama (phi model) for biological insights.
